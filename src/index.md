# Übersicht

In dieser Dokumentation lernen Sie, wie Sie Seatsurfing auf Ihrem Server einrichten, als Organisations-Administrator Sitzplätze einrichten und als Mitarbeiter die mobile App auf Ihrem Smartphone nutzen.
